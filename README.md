# Inventory and Sales Management System with Inventory Analytics

## Project Overview
This project is a **full-stack Inventory and Sales Management System** designed to help small and medium businesses manage their inventory efficiently.  
It integrates product management, sales transactions, automatic stock updates, and inventory-focused analytics in a single platform.

The system helps businesses avoid:
- Stock shortages
- Excess inventory
- Dead stock
- Poor inventory visibility

## Features

### Product Management
- Add products
- Delete products
- Manage stock levels
- Set reorder levels

### Sales Management
- Record sales transactions
- Multiple items per sale
- Automatic stock deduction

### Inventory Analytics
- Low stock detection
- Dead stock identification
- Stock movement tracking
- Inventory dashboard

---

## System Workflow

User Login  
→ Add / Update Products  
→ Set Stock & Reorder Levels  
→ Record Sales Transaction  
→ Automatic Stock Update  
→ Inventory Analysis  
→ Generate Inventory Reports
